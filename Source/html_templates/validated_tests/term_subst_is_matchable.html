Tests to validate for the function Term.Subst.is_matchable

Given two lists l1 = \( [u_1;\ldots;u_n]\) and l2 = \( [v_1;\ldots;v_n]\), is_matchable at l1 l2 returns \(\top\) iff there exists a substitution \(\sigma\) such that \(\forall i \in \mathbb{N}^n_1, u_i\sigma = v_i\). Note that we allow \(\sigma\) to be cyclic and to not respect types (for second-order variables).

We also adopt the following convention:

  • Free first-order (resp. second-order variables) will be denoted by \(w\) (resp. \(W\) )
  • Existential first-order (resp. second-order variables) will be denoted by \(x\) or \(y\) (resp. \(X\) or \(Y\) )
  • Universal first-order (resp. second-order variables) will be denoted by \(z\) (resp. \(Z\) )
  • Public names will be denoted by \(a\) or \(b\) or \(c\)
  • Bound names will be denoted by \(k\) or \(l\) or \(m\)
