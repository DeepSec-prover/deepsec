Tests to validate for the function Term.Modulo.syntactic_equations_of_equations

syntactic_equations_of_equations l_eq returns a set of substitutions that can be seen as a disjunction of substitution \( \sigma_1 \vee \ldots \vee \sigma_n\) such that if l_eq is the list of equations corresponding to \(\bigwedge_{i=1}^m u_i =_{\mathcal{R}} v_i\) and \(V = \bigcup_{i=1}^m \mathit{vars}^1(u_i,v_i)\) then for all \(\sigma\), \[ \sigma \models \bigwedge_{i=1}^m u_i =_{\mathcal{R}} v_i \mbox{ iff } \sigma \models \bigvee_{i=1}^n \exists E_i. \bigwedge_{ x \in \mathit{dom}(\sigma_i)} x =^? x\sigma_i \] where for all \(i \in {1,\ldots, n}, E_i = \mathit{vars}^1(\sigma_i) \setminus V\). Note that all the new variables in the different substitutions are all existentially fresh.

If the list of equations is always satisfiable (resp. unsatisfiable) then \(\top\) (resp. \(\bot\)) will be displayed as result.

Note that we adopt the following convention:

  • Free first-order (resp. second-order variables) will be denoted by \(w\) (resp. \(W\) )
  • Existential first-order (resp. second-order variables) will be denoted by \(x\) or \(y\) (resp. \(X\) or \(Y\) )
  • Universal first-order (resp. second-order variables) will be denoted by \(z\) (resp. \(Z\) )
  • Public names will be denoted by \(a\) or \(b\) or \(c\)
  • Bound names will be denoted by \(k\) or \(l\) or \(m\)
